# The Endless Halls

A simple browser-based game built with HTML, JavaScript, and CSS.

## How to Play

1. Download and extract the `.zip` file.
2. Open the `index.html` file in your web browser.
3. Use the arrow keys or WASD to move your character, and press Space or Enter to interact.

## Controls
- **Arrow keys / WASD**: Move the player.
- **Space / Enter**: Interact with objects.

Enjoy the game!
